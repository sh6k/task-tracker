# Task Tracker CLI

Project URL: https://roadmap.sh/projects/fitness-workout-tracker 
A CLI task manager built in C++ with no external dependencies.
Tasks are stored in `tasks.json`.

## Build
Open in Visual Studio and build with Ctrl+Shift+B.

## Usage
```cmd
"Task Tracker.exe" add "Buy groceries"
"Task Tracker.exe" list
"Task Tracker.exe" mark-done 1
"Task Tracker.exe" delete 1
```
